# 8-Bit Tees

A custom web application for retro 8-bit style t-shirts targeting the Solana DeFi community.

## Target Audience
**Solana Degens** - Active Solana DeFi traders and community members who appreciate:
- Retro 8-bit gaming aesthetics
- Crypto culture and memes
- High-quality streetwear
- Exclusive/limited drops

## Vision
Connect nostalgic 8-bit gaming vibes with modern Solana culture through premium apparel that degens actually want to wear and show off.
